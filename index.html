Understanding BeePrint Project

Why bees?

Bees are the key in order to sustain the flora of an ecosystem.
They help pollinate the flowers so that plants can multiply each season and can satisfy the demand of herbivorous animals.

Benefits of understanding their movement

Ecosystems are prone to degenerate in today's industrial age. We want to study bees to prevent that. Our project aims to understand what makes bees behave the way they do so that in near future if we need random bees to form groups and pollinate barren lands we would know what kind of stimilus we would need.

Tools we used

We can divide the tools we used into 2 categories. Hardware and software. We used arduino as hardware to automate the electrical grid experiments we used. Bees are given electric pulses in order to move around the grind. We also tagged the bees with QR codes on their backs to see them on the monitors. We used various software tools to detect bees, measure their movements and apply statistics to them and of course building a website. In the end we aim to make researchers have a tool to understand bees in a comprehensive way.

Hardware Software

Hardware: Arduino

Arduino is an open-source electronics platform based on easy-to-use hardware and software. Arduino boards are able to read inputs such as light on a sensor or a finger on a button and turn it into an output like activating a motor turning on an LED. We used arduino in our grid experiments. It controls the electric pulses we give to the grid and make bees move.

SOFTWARE

Website design

Django is a high-level Python web framework that enables rapid development of secure and maintainable websites. We used it to create our front-end and back-end development of our project's website.

Website development

React allows developers to create large web applications that can change data, without reloading the page. We used it for our user-interfaces. We aim to make user-friendly website which is easy to use and understand.

Image Handling

We tagged the bees with QR codes on their backs. We used openCV to detect them virtually on the monitor. We also used openCV to apply homography to normalize the videos to perform more accurate statistical analysis.

Database manipulation

We need databases to store the experiments' data. We used sqLite to perform database functions.
